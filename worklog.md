## 🔄 تدفق عمل كل خدمة

### 1️⃣ تتبع الطلب (Order Tracking)
```
المستخدم ← يفتح الخدمة ← يبحث برقم الطلب ← يرى تقدم الطلب في 6 مراحل:
pending → approved → documents → final_approval → contract → delivery
```

### 2️⃣ طلب سيارة جديدة (New Car Request)
```
المستخدم ← يملأ نموذج مفصل (السيارة + البيانات الشخصية + التوظيف) ← يقدم الطلب ← يحصل على رقم تتبع
```

### 3️⃣ أسعار السوق (Market Prices)
```
المستخدم ← يختار الماركة والموديل ← يرى جدول مقارنة الأسعار من مصادر متعددة
```

### 4️⃣ العروض (Offers)
```
المستخدم ← يتصفح العروض المتاحة ← يختار عرض ← يحصل على كوبون خصم
```

### 5️⃣ الضمان الممتد (Extended Warranty)
```
المستخدم ← يختار باقة الضمان ← يرى التفاصيل والتسعير ← يقدم طلب
```

### 6️⃣ المقارنة (Compare)
```
المستخدم ← يضيف سيارات (حتى 4) ← يرى جدول مقارنة شامل ← يضيف للمقارنة من المنافسين تلقائياً
```

### 7️⃣ الحسبة/التمويل (Financing)
```
المستخدم ← يدخل الراتب والدفعة المقدمة ← يختار المدة (1-5 سنوات) ← يرى:
- القسط الشهري
- عروض البنوك السعودية
- برامج التمويل (مرابحة، تمليك، تقسيط، تأجير)
- الأهلية حسب الراتب
```

### 8️⃣ الصيانة (Maintenance)
```
المستخدم ← يختار نوع السيارة ← يرى جدول صيانة شامل مع التنبيهات
```

### 9️⃣ تقييم السيارة (Car Valuation)
```
المستخدم ← يدخل بيانات سيارته (الماركة، الموديل، السنة، الممشى، الحالة) ← يحصل على تقييم السوق
```

### 🔟 الإعدادات (Settings)
```
المستخدم ← يتحكم في: اللغة، الثيم، الإشعارات ← يصل لسياسة الخصوصية وشروط الخدمة
```
