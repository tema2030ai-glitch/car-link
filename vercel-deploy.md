# 🚀 خطوات نشر المشروع على Vercel

## الطريقة 1: النشر المباشر من GitHub
1. اذهب إلى github.com وأنشئ repository جديد
2. ارفع المشروع:
   git remote add origin https://github.com/USERNAME/car-link.git
   git push -u origin main

3. اذهب إلى vercel.com
4. اضغط "Sign up" ثم "Continue with GitHub"
5. اضغط "New Project"
6. اختر repository الخاص بك
7. اضغط "Deploy"

## الطريقة 2: النشر المباشر (بدون GitHub)
1. ثبّت Vercel CLI: npm i -g vercel
2. شغّل: vercel
3. اتبع التعليمات

## النتيجة:
ستحصل على رابط مثل: https://car-link-xxx.vercel.app
